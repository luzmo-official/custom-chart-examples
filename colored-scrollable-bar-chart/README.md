# Custom Chart – Colored scrollable bars

The **colored scrollable bars** chart allows you to maintain readability on bar charts with a large number of categories while also providing color coding for values from 0 to 100.

<video muted autoplay loop src="https://github.com/luzmo-official/custom-chart-examples/raw/refs/heads/main/colored-scrollable-bar-chart/colored_scrollable_bars_preview.mp4" width="800" /> 

## Using in Luzmo

Upload `bundle.zip` in the Luzmo app in [your Profile settings -> Custom charts](https://app.luzmo.com/settings/custom-charts)

## Development

See Custom Chart Builder repository [luzmo-official/custom-chart-builder](https://github.com/luzmo-official/custom-chart-builder). 

You'll want to copy-paste the `package.json` file and the `src` directory from here into `/projects/custom-chart` of that repository (replacing all existing files).
